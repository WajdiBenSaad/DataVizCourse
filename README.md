## Contributing

Contributions and feedback are welcome! Below are simple ways to help and how to submit changes.

### Ways to contribute
- Report bugs or open an issue: https://github.com/WajdiBenSaad/DataVizCourse/issues
- Suggest new topics, improvements, or examples
- Fix typos, formatting, or documentation errors via pull request
- Add examples, notebooks, or resources (place them in the appropriate folder and update the README)
- Share the course with colleagues and on social platforms

### How to submit changes
1. Fork the repository and create a feature branch.
2. Make changes, add tests/examples if relevant, and keep commits focused and descriptive.
3. Open a pull request that references any related issue and describes the change.
4. Respond to review comments and update the PR as needed.
